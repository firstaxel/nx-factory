---
title: add-component
description: Add shadcn components to the shared UI package.
---

## Usage

```bash
nx-factory add-component
nx-factory add-component button card dialog
nx-factory add-component data-table --dry-run
```

## Arguments

| Argument | Description |
| --- | --- |
| `[components...]` | Optional component names |

## Options

| Option | Description |
| --- | --- |
| `--dry-run` | Preview changes only |

## What It Does

- Adds component files to shared UI package
- Updates UI barrel exports automatically
