---
title: add-component
description: Add shadcn/ui components to the shared UI package and automatically update the barrel export.
---

## Usage

```bash
nx-factory-cli add-component
nx-factory-cli add-component button card dialog
nx-factory-cli add-component data-table --dry-run
```

## Arguments

Pass one or more component names as arguments, or run with no arguments for an interactive checkbox picker showing all 47 available shadcn/ui components.

## Options

| Option | Description |
|---|---|
| `--dry-run` | Preview what would be added |

## What it does

1. Detects the UI package directory (looks for `components.json`)
2. Runs `shadcn@latest add --yes <components>` inside `packages/ui`
3. Updates `src/index.tsx` (or `index.tsx`) to re-export the new components

After running, any app in the monorepo can import the component immediately:

```tsx
import { Button, Card } from "@workspace/ui";
```

No rebuild required for development — the workspace symlink resolves to the source directly.
