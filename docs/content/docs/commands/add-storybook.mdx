---
title: add-storybook
description: Add Storybook to the shared UI package.
---

## Usage

```bash
nx-factory add-storybook
nx-factory add-storybook --dry-run
```

## Options

| Option | Description |
| --- | --- |
| `--dry-run` | Preview setup without writing |

## What It Does

- Adds Storybook config for shared UI package
- Generates starter stories for components where supported
