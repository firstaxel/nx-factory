## Usage

```bash
nx-factory-cli doctor
```

## What it checks

- **tsconfig inheritance** — all packages and apps extend `../../tsconfig.base.json`
- **@scope/* paths** — `tsconfig.base.json` has the correct paths mapping
- **Barrel exports** — `packages/ui/index.tsx` exports all installed shadcn/ui components
- **package.json exports** — the UI package has a well-formed `exports` field
- **nx-factory.config.json** — present and readable

Auto-fixes barrel export issues when found. Reports all other issues without modifying files.
