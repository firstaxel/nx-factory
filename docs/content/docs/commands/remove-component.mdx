---
title: remove-component
description: Remove shadcn components from shared UI package.
---

## Usage

```bash
nx-factory-cli remove-component
nx-factory-cli remove-component button dialog
nx-factory-cli remove-component tooltip --yes
nx-factory-cli remove-component badge --dry-run
```

## Arguments

| Argument | Description |
| --- | --- |
| `[components...]` | Optional component names |

## Options

| Option | Description |
| --- | --- |
| `-y, --yes` | Skip confirmation prompts |
| `--dry-run` | Preview changes only |

## What It Does

- Removes component files
- Cleans barrel exports
