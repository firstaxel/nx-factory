---
title: update
description: Update installed shadcn components.
---

## Usage

```bash
nx-factory-cli update
nx-factory-cli update button card
nx-factory-cli update --yes
nx-factory-cli update --dry-run
```

## Arguments

| Argument | Description |
| --- | --- |
| `[components...]` | Specific components to update (default: all) |

## Options

| Option | Description |
| --- | --- |
| `-y, --yes` | Skip confirmation |
| `--dry-run` | Preview changes only |

## What It Does

- Updates selected or all shadcn components
- Applies component template refreshes
